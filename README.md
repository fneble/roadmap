How to use it
Open the link above in any browser. No login required.
Editing:

Click any text to edit it inline
Drag cards left or right to reorder them within a pillar
Use the + button at the bottom of a pillar to add a new card
Use the × button on a card to remove it
Add or remove subpoints, dependency chips, assignees, and links within each card

Changes are saved automatically in your browser (via localStorage). They persist across sessions but are local to your device — edits made by one person won't appear for others.

Updating the shared version
Since edits are per-device, the recommended workflow for keeping a shared source of truth is:

Make your changes in the browser
Export the updated HTML (or edit roadmap.html directly)
Upload the new file to this repo — GitHub Pages will redeploy automatically within ~60 seconds


File structure
roadmap.html    # The entire roadmap — one self-contained file
README.md       # This file

Contributing

Make edits to roadmap.html
Open a pull request, or push directly to main if you have access
The live site updates automatically after ~60 seconds


Built with plain HTML, CSS, and JavaScript. No frameworks or dependencies.
